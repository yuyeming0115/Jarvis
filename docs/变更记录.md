# 变更记录

## 2026-07-03

### V1.2 SQLite 数据库版

- 新增 SQLite schema 和 `backend/db/jarvis.sqlite3` 主数据源。
- 后端数据层从 JSON 读写升级为 SQLite 读写。
- 首次初始化会从现有 JSON 数据迁移到 SQLite。
- 新增 `init-sqlite.sh`、`export-json.sh`、`backup-db.sh`。
- `backup-json.sh` 改为从 SQLite 导出 JSON。
- 写入前自动备份 SQLite，并导出 JSON 快照。

### V1.1 本地可编辑版

- 新增 Python 标准库后端 API，不安装外部依赖。
- 工作台改为通过 `/api` 读取和写入 JSON。
- 支持新增任务、灵感、选题。
- 支持修改任务状态、优先级和标记完成。
- 后端写入前自动创建 JSON 快照备份。
- 新增 API 冒烟测试脚本，支持只读检查和可选写入检查。
- 新增备份与 Git 策略文档。
- 新增 `.gitignore`，排除真实密钥、本地 JSON 数据、日志、PID、备份和数据库文件。

### V1.0 本地展示版

- 创建 Jarvis V1.0 本地展示版目录结构。
- 创建工作台静态页面、样式和数据读取脚本。
- 创建本地 JSON 示例数据：任务、灵感、选题、系统状态、运行日志。
- 创建启动、停止、健康检查和 JSON 备份脚本。
- 启动脚本会等待本地 HTTP 服务可用；停止脚本只停止已记录的 Jarvis 工作台进程。
- 创建 `.env.example` 和 launchd 示例文件。
- 创建运行手册、数据结构、多入口接入策略和安全边界文档。
