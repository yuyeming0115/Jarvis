# 数据结构

## tasks.json

任务字段包括：

- `task_id`
- `title`
- `description`
- `project`
- `source`
- `due_at`
- `priority`
- `status`
- `reminder_level`
- `tags`
- `sync_status`
- `created_at`
- `updated_at`

## ideas.json

灵感字段包括：

- `idea_id`
- `raw_text`
- `type`
- `tags`
- `status`
- `ai_summary`
- `source`
- `sync_status`
- `created_at`
- `updated_at`

## topics.json

选题字段包括：

- `topic_id`
- `title`
- `angle`
- `platform`
- `content_type`
- `target_audience`
- `score`
- `status`
- `draft_status`
- `source`
- `sync_status`
- `created_at`
- `updated_at`

## system-status.json

记录本地工作台、数据源、外部入口、安全模式和公网访问状态。

## logs.json

记录初始化、后续操作、失败和重试事件。V1.0 只写入初始化日志。

V1.1 开始，新增、更新、完成任务等操作会写入运行日志。
