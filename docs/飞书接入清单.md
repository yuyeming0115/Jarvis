# 飞书接入清单

## 当前 V2.0 已完成

- 本地事件回调接口：`POST /api/feishu/event`
- 本地 URL 验证 challenge
- 明文文本消息解析
- 规则分类入库
- 消息记录表 `messages`
- 本地模拟脚本

## 你需要手动准备

1. 登录飞书开放平台。
2. 创建自建应用或机器人应用。
3. 获取 `APP_ID`、`APP_SECRET`、`Verification Token`。
4. 先不要配置 `Encrypt Key`，V2.0 本地适配器暂不解密加密事件。
5. 准备一个安全的 HTTPS 回调地址。

## 本地配置

真实密钥只放在本机 `.env`，不要粘贴到聊天窗口或 Git：

```bash
cp "$HOME/Jarvis/adapters/feishu/feishu.env.example" "$HOME/Jarvis/adapters/feishu/feishu.env"
```

## 安全边界

- 不暴露 Mac Mini 管理端口。
- 不直接把 `127.0.0.1:8080` 暴露公网。
- 不抓取公司飞书或公司系统数据。
- 只处理你发给 Jarvis 机器人的消息。
- 外部回调方案需要单独确认后再启用。
